# CreditWise Loan Approval Prediction System

## 📌 Project Overview
This project predicts whether a loan application will be approved using Machine Learning classification algorithms. It includes data preprocessing, feature engineering, model training, and evaluation.

## 🚀 Features
- Data Cleaning & Preprocessing
- Feature Engineering
- Model Comparison
- Performance Evaluation

## 🛠 Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Jupyter Notebook

## 📂 Dataset Features
- Applicant Income
- Coapplicant Income
- Credit Score
- Loan Amount
- Loan Term
- DTI Ratio
- Savings
- Existing Loans
- Employment Type
- Marital Status

## ⚙️ Feature Engineering
- DTI_Ratio_sq
- Credit_Score_sq
- Applicant_Income_log

## 🤖 Machine Learning Models
- Logistic Regression
- K-Nearest Neighbors (KNN)
- Gaussian Naive Bayes

## 📊 Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1 Score
- Confusion Matrix
